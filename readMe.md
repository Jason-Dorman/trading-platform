##### **SetUp**
   Install homebrew:
   ` /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"`
    
   Activate virtual env using python 3.8:
    `conda activate wolf`
    
   Install pyenv:
    `brew install pyenv`
    
   Install python 3.8 for interpreter:
    `pyenv install 3.8.5`
